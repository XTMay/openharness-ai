# Code of Conduct

OpenHarness AI is built for respectful, practical collaboration.

Participants are expected to:

- Be kind and professional.
- Assume good intent, but clarify when something is unclear.
- Keep technical disagreement focused on the work.
- Avoid harassment, insults, or exclusionary behavior.
- Respect maintainers' decisions about project scope and safety.

Maintainers may remove comments, issues, pull requests, or contributors that violate these expectations.
